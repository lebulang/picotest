Wave VR SDK Developer Guide¶

Wave VR is an API and runtime which enables developers to create VR contents on Android™. Our aim is to provide a flexible and simple interface for developers so that developers can focus on designing awesome and immersive VR experiences without requiring specific knowledge of the VR hardware.

Wave VR SDK contains the APIs, libraries, and samples for developing VR apps. This guide shows you the following contents.
